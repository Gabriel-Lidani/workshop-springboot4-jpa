# Sobre o projeto

Este projeto consiste em uma API REST para gerenciamento de usuários, pedidos e produtos, implementando relacionamentos entre entidades e persistência de dados utilizando JPA/Hibernate.

A aplicação simula um sistema de e-commerce, permitindo operações básicas de consulta e manipulação de dados.

# Como executar o projeto
Pré-requisitos: Java 17

```bash
# clonar repositório
git clone https://github.com/Gabriel-Lidani/workshop-springboot4-jpa

# entrar na pasta
cd workshop-springboot4-jpa

# executar projeto
./mvnw spring-boot:run
```
